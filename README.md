# CyberArena — сайт компьютерного клуба

Итоговый проект по дисциплине **«Проектирование информационных систем с применением WEB-технологий»**.

Тема проекта: **создание и реализация сайта с базой данных «Компьютерный клуб» — клиентская часть и административная панель**.

## Состав проекта

В репозитории размещён архив:

* `Проект.zip` — полный комплект файлов сайта и базы данных.

Внутри архива находятся:

* клиентская часть сайта;
* административная панель;
* подключение к базе данных MySQL;
* SQL-файл базы данных;
* служебные файлы проекта.

## Основной функционал

### Клиентская часть

* просмотр главной страницы и каталога игровых мест;
* фильтрация игровых мест;
* регистрация и авторизация пользователя;
* добавление игровых мест в корзину;
* оформление бронирования;
* просмотр своих бронирований;
* личный кабинет с редактированием данных;
* форма обратной связи;
* счётчик посещений сайта.

### Административная часть

* отдельный вход администратора;
* управление пользователями;
* просмотр и удаление бронирований;
* обработка заявок восстановления доступа;
* добавление, редактирование и удаление игровых мест.

## Используемые технологии

* HTML;
* CSS;
* JavaScript;
* PHP;
* MySQL;
* phpMyAdmin;
* Open Server.
